# 当前最优配方（交付）

更新日期: 2026-06-11

## 单场景识别（注册与验证同一次采集）—— 推荐

**配方: V21 + V18b 全局嵌入融合 + T-norm**
- EER = **1.10% ± 0.006%**（3次模板采样，极稳定）
- 优于 V21 单独的 1.19%，且消除了 V21 随模板选择的剧烈波动（1.2~2.1%）
- 零额外训练，纯推理期融合

**实现**（见 `fusion_final.py` / `final_confirm.py`）:
1. 对每帧, 用 `models_v21/v21_seed{42,123,777}.pth`(HybridDelfEncoder) 和
   `models_v18b/v18b_seed{42,123,777}.pth`(DenseDescriptorEncoder) 各提 512维全局嵌入
2. 每个模型: 双极性(±)前向 → 平均 → L2归一化
3. 6个模型嵌入等权平均 → L2归一化 = 融合嵌入
4. 匹配分数 = max over 20注册模板的余弦相似度
5. T-norm: `t = (score - mean(impostor_scores)) / std(impostor_scores)`

## 跨场景识别（注册后换次采集再验证）—— 部署推荐

**配方: V25a（同手指合并训练）**
- EER ≈ **5.7%**（跨场景），相比 V21 冻结的 57%（≈随机）是质变
- 融合 V25a+V26b 在噪声带内无稳健改善（6.3%±0.6%）
- 模型: `models_v25a/v25a_seed{42,123,777}.pth`
