## 一、本阶段结论摘要（一页看懂）

| 项目 | 结论 |
|------|------|
| **贴屏精度** | ❌ 未达标：纹理法 EER 17–20%、CNN 法 12–16%；严格 FAR<1/5W 时 FRR ≥54%（目标 <3%） |
| **根因** | 贴屏小面积超声 + 跨场景再采集 → 同指帧间**重叠低**、噪声大；纹理法/CNN 两大类殊途同归，**是数据侧问题，非算法选择** |
| **top1000 质量筛选** | 原图略优于降噪，但仍不达标（FAR=0 FRR 58%）；**降噪有害，用原图** |
| **模板数量** | 实测**越多越好**（K=20 最优），不能减 |
| **旧数据(单场景)参考** | 已达标：FAR=0 时 FRR 2.34%（全 27 指）/ 0.64%（25 指） |

**后续待客户拍板**：① 严格 FAR<1/5W 是否可放宽；② 能否提供更多贴屏跨场景配对数据用于正式重训。

## 六、方法要点（两条核心设计）
1. **真实对应点重训**：同指两帧对齐 → RANSAC 内点对 = 同一物理脊线点的对应位置 → 取 32×32 patch 作正样本对重训（比纯增广更贴近真实跨帧变化）。
2. **重叠面积归一化**：A 前景按位姿 warp 到 B ∩ B 前景 = 重叠面积；分数 = 内点数 × √(20000/重叠面积)。

## 七、后续（贴屏数据方向，待与客户确认）
