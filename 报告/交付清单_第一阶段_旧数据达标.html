日期：2026-07-01 范围：老数据（无贴屏 + 不贴屏，27 类手指）上的指纹匹配精度达标——纯纹理方向（禁 SIFT）。 说明：本包仅含第一阶段（旧数据）内容；贴屏/新数据实验、部署空间时间优化均不在此包。

一、达标结论（一页看懂）

目标：FAR < 0.002%（1/5万）时 FRR < 3%。

范围 EER FAR=0 时 FRR 结论
全 27 根手指 1.83% 2.34% ✅ 达标（<3%）
排除 1 根采集退化手指（wtp_xyz_R1） 1.00% 0.64% ✅ 优于参考方案 0.96%
  • 冒充被彻底压制：冒充的几何一致内点 max ≈ 19，FAR=0 完全可达。
  • 零 SIFT：纯自训纹理描述子。
  • 2 根手指（btp_fys_R0 / wtp_xyz_R1）的 test 段为采集退化（注册段匹配良好、test 帧内点≈0），SIFT / CNN / 纹理三方交叉确认是数据损坏，卡全集下限；25 根干净手指达标充分。

二、方法（纹理法，三步）

  1. 局部纹理描述子（HardNet 风格自训）：32×32 脊线 patch → 128 维 L2 描述子；真实跨帧对应点重训（同指两帧对齐的 RANSAC 内点对作正样本，比纯增广更贴近真实变化），可 bootstrap。
  2. 几何验证 = 姿态对齐：密集网格（stride 6）提描述子 → 互最近邻 + ratio 匹配 → RANSAC 估位姿 → 变换约束（scale∈[0.85,1.18]、|rot|<30°）压制冒充 → 几何一致内点数为匹配分。
  3. 重叠面积归一化：按对齐后前景重叠面积归一化内点数，使不同重叠的分数可比。

其它路线（同为老数据）：两级全局+局部（FAR=0 时 FRR 2.1%/25 根）；CNN 全局嵌入 V21+V18b 融合（单场景 EER 1.10%，但连续分数尾部重叠、达不到严格 FAR——严格 FAR 需内点几何的离散可分分布）。
