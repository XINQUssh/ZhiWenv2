## 3. 各版本详细分析

### 3.1 V14 — 基线 (EER = 2.6776%)

**架构**: ResNet-18 (ImageNet预训练) + GAP + Linear → 512维嵌入
**训练**: SimCLR 80ep（冻结backbone）+ ArcFace 120ep
**推理**: 双极性TTA，余弦相似度

| 指标 | 值 |
|------|-----|
| EER | 2.6776% |
| FFR=3% → FAR | ~1.5% |
| FFR=5% → FAR | ~0.5% |

**问题**: 无SWA导致过拟合；无分数归一化；简单模板选择

---

### 3.2 V15 — SWA引入 (EER = 2.40%)

**改进**: 在V14基础上加入SWA（最后20个epoch权重平均）
**结果**: EER从2.68%降至2.40%，确认SWA对小数据集有效

---

### 3.3 V16 — 复杂架构尝试 (EER = 2.5144%)

**架构**: ConvNeXt-Tiny (28.6M参数) + 17通道Gabor输入 + GeM池化 + 多尺度特征聚合
**训练**: SimCLR 60ep + ArcFace 140ep + Hard Triplet 30ep（3阶段）

| 配置 | Backbone | EER |
|------|----------|-----|
| Config A（主） | ConvNeXt-Tiny | 2.5144% |
| Config C（对照） | ResNet-18 | 3.7870% |

**结论**: 复杂架构在27类小数据集上反而过拟合，ConvNeXt的28.6M参数远超数据支撑能力。**简单架构优于复杂架构**。

### 3.4 V17 — 分数归一化突破 (EER = 1.6416%)

**核心创新**: T-norm分数归一化 — 零训练开销的后处理方法

**架构**: 与V14相同的ResNet-18 + GAP + Linear → 512维
**训练**: SimCLR 80ep（冻结）+ ArcFace 140ep（SWA@110, label_smoothing=0.1）
**参数量**: 11,433,920
**推理**: 4-view TTA (2极性 × 2位移) + 3模型集成(seeds 42/123/777)

**T-norm原理**:
```
对于probe与claimed identity的raw_score:
impostor_scores = probe与所有其他identity的分数
t_score = (raw_score - mean(impostor_scores)) / std(impostor_scores)
```

#### 所有变体EER

| 模板策略 | raw | Z-norm | T-norm | S-norm |
|----------|-----|--------|--------|--------|
| quality (质量筛选) | 2.19% | 2.38% | 2.48% | 2.32% |
| random (随机20帧) | 2.22% | 1.76% | **1.64%** | 1.96% |

**关键发现**: Random模板 + T-norm效果最好。Quality模板筛选的模板间多样性不足，反而损害T-norm效果。

### 3.5 V18 — 未训练密集局部匹配 (EER = 1.6865%)

**核心思路**: 借鉴FLARE/DMD的空间描述符方法，复用V17模型的Layer3特征图(256, 14, 13)作为密集局部描述符，无需重新训练。

**密集匹配方法**:
- 每帧产生182个空间位置(14×13)，每个位置256维描述符
- 双向max-mean匹配：query的每个位置找template中最佳匹配，取top-70%均值
- 融合: `fused = α × dense_score + (1-α) × global_score`

#### 所有变体EER

| 变体 | EER |
|------|-----|
| global_raw | 2.08% |
| global_tnorm | 2.07% |
| dense_raw | 8.89% |
| dense_tnorm | 7.17% |
| fused_a0.3_tnorm | **1.69%** |
| fused_a0.5_tnorm | 1.77% |
| fused_a0.7_tnorm | 1.75% |

**关键结论**:
- Dense单独匹配极差（EER=8.89%），因为Layer3特征是为全局分类优化的，单个空间位置的特征不具备局部判别性
- 融合后仅微弱改善（1.69% vs V17的1.64%），说明未训练的局部特征贡献有限
- **验证了需要专门训练局部描述符的必要性**

### 3.6 V18b — 训练局部描述符 (EER = 1.3341%) ⬅ 当前最优

**核心创新**: 在ResNet-18的Layer3后增加专用局部描述符头，通过位置对比学习联合训练

**模型架构**:
```
ResNet-18 backbone
    ├─ Layer3 (256, 14, 13)
    │    └─ local_head: Conv1x1(256→64) + BN + L2-norm → 局部描述符 (64, 14, 13)
    └─ Layer4 → GAP → Linear+BN → 512维全局嵌入
```

**训练策略**:
- Phase 1: SimCLR + PositionContrastiveLoss (60 epochs, backbone冻结)
- Phase 2: ArcFace + PositionContrastiveLoss (120 epochs, SWA@100)
- 联合损失: `loss = loss_arc + 0.3 × loss_local`
- 参数量: 11,450,432（仅增加~17K参数的local_head）

**PositionContrastiveLoss原理**:
```
同一帧的两个增强视图 → 各自产生 (64, 14, 13) 描述符图
对应空间位置(i,j)为正样本对
不同位置为负样本对
InfoNCE loss: 使每个位置的描述符具有唯一判别性
```

**训练时间**: 3个seed共约145分钟

## 4. 核心发现与经验教训

### 4.1 有效的技术

| 技术 | EER改进 | 额外训练开销 | 推荐度 |
|------|---------|-------------|--------|
| T-norm分数归一化 | -0.58% (2.22→1.64) | 零 | ★★★★★ |
| SWA权重平均 | -0.28% (2.68→2.40) | 零 | ★★★★★ |
| Label Smoothing (0.1) | ~-0.1% | 零 | ★★★★ |
| 位置对比学习(局部描述符) | -0.31% (1.64→1.33) | 中等 | ★★★★ |
| 全局-局部融合 | -0.07% (1.41→1.33) | 推理时 | ★★★ |
| 4-view TTA | ~-0.1% | 推理4x | ★★★ |
| 3模型集成 | ~-0.2% | 训练3x | ★★★ |

### 4.2 无效/负面的技术

| 技术 | 结果 | 原因分析 |
|------|------|----------|
| ConvNeXt-Tiny (28.6M参数) | 比ResNet-18差 | 27类小数据集严重过拟合 |
| Gabor滤波器输入 | 无明显改善 | CNN自身可学习类似滤波器 |
| GeM池化 | 无明显改善 | 数据量不足以训练额外参数 |
| 多尺度特征聚合 | 无明显改善 | 增加复杂度但无收益 |
| Quality模板筛选 | 反而损害T-norm | 模板多样性降低影响归一化 |
| Hard Triplet Mining (Phase 3) | 无明显改善 | ArcFace+SWA已足够 |
| 未训练的dense局部匹配 | 几乎无改善 | 全局特征不适合局部判别 |

### 4.3 关键结论

1. **简单架构 + 强正则化 > 复杂架构**: ResNet-18 (11.4M) 始终优于ConvNeXt-Tiny (28.6M)
2. **后处理 > 模型改进**: T-norm的0.58%改进大于任何模型架构改动
3. **专门训练的局部特征有价值**: 但必须通过位置对比学习专门训练，不能复用全局特征
4. **问题手指稳定存在**: wtp_xyz_R1、btp_fys_R0始终是最差手指，可能是采集质量问题

## 5. 距离目标的差距分析

**客户目标**: FAR < 0.002% (1/50000) 时 FFR < 3%

**当前最优 (V18b fused_a0.5_tnorm)**:
- FFR=3% → FAR=0.32%（需要 < 0.002%，差160倍）
- FAR=0.002% → FFR=100%（需要 < 3%）

| 从不同角度看差距 | 当前值 | 目标值 | 差距 |
|-----------------|--------|--------|------|
| FFR=3%时的FAR | 0.32% | 0.002% | 160× |
| FAR=0.002%时的FFR | 100% | <3% | 极大 |
| 需要的d-prime估算 | 5.20 | >8.0 | +2.8 |

**差距本质**:
- 当前impostor分布的尾部与genuine分布仍有重叠
- 在21,138个impostor对中，最高impostor分数接近genuine均值
- 需要将impostor的最大值从8.70降低，或将genuine的最小值从-1.33提升
